    table-reader plugin

    This site serves as an example of how to use the table-reader plugin.

    read_csv

    The table below was inserted using the {{ read_csv('tables/basic_table.csv') }} tag:

    a b
    40 73
    50 52
    531456 80
    name table1

    read_excel

    The table below was inserted using the {{ read_excel('tables/excel_table.xlsx') }} tag:

    a b
    40 73
    50 52
    531456 8

    Reading xlsx files

    You might get a XLRDError('Excel xlsx file; not supported',) error on linux systems. That's because xlrd does not support .xlsx files (stackoverflow post). Instead, install openpyxl and use:

    {{ read_excel('tables/excel_table.xlsx', engine='openpyxl') }}
